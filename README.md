# express-mvc-framework

This open-source MVC framework is written in Javascript and built on ExpressJS. 

## Requirements

node.js - LTS version 16 or higher

## Installation

Install dependencies
```bash
npm install
```
Install Dynamodb
```bash
serverless dynamodb install
```

## Run the application

Run DynamoDB database locally
```bash
serverless dynamodb start
```

Run service locally
```bash
npm run dev
```
Go to localhost:8080 in your browser.

## Test the application

Run DynamoDB database locally
```bash
serverless dynamodb start
```

Run test
```bash
npm run test
```
